# 音声・効果音（BGM & SE）ファイル配置フォルダ

このフォルダ (`public/sounds/`) 内に以下のファイル名で音声ファイルを配置してください。

## 音声ファイル仕様

| ファイル名 | 説明 | 形式 | 備考 |
| :--- | :--- | :--- | :--- |
| **`bgm.mp3`** | ゲーム全体でループ再生されるBGM（背景音楽） | `.mp3` / `.wav` / `.ogg` | ループ再生されます |
| **`shoot.mp3`** | プレイヤーが弾を発射したときの効果音 | `.mp3` / `.wav` / `.ogg` | 発射の瞬間に再生 |
| **`defeat.mp3`** | 敵キャラクターが倒れた（撃破された）ときの効果音 | `.mp3` / `.wav` / `.ogg` | 撃破の瞬間に再生 |
| **`enemy_shoot.mp3`** | 敵キャラクターが弾を発射したときの効果音 | `.mp3` / `.wav` / `.ogg` | 敵の攻撃時に再生 |

※ 画面右上の **スピーカーアイコン (🔊 / 🔇)** から、BGMとSEの音量調節（0%〜100%）およびON/OFF（ミュート）を切り替えることができます。
※ ファイルが存在しない場合でもエラーでゲームが停止しない安全設計になっています。
