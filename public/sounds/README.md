# 効果音（SE）ファイル配置フォルダ

このフォルダ (`public/sounds/`) 内に以下のファイル名で効果音ファイルを配置してください。

## 推奨ファイル名

| ファイル名 | 説明 | 形式 |
| :--- | :--- | :--- |
| **`shoot.mp3`** | プレイヤーが弾を発射したときの効果音 | `.mp3` / `.wav` / `.ogg` |
| **`defeat.mp3`** | 敵キャラクターが倒れた（撃破された）ときの効果音 | `.mp3` / `.wav` / `.ogg` |
| **`enemy_shoot.mp3`** | 敵キャラクターが弾を発射したときの効果音 | `.mp3` / `.wav` / `.ogg` |

※ ファイルが存在しない場合でもエラーでゲームが停止しない安全設計になっています。
